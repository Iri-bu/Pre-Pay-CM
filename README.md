# Прототип настройки OTA-предоплат

Интерактивный HTML-прототип страницы личного кабинета TravelLine:
`Менеджер каналов -> Настройки -> Предоплаты из OTA`.

## Что внутри

- `index.html` — самодостаточный прототип без внешних зависимостей.
- `.nojekyll` — служебный файл для корректной публикации через GitHub Pages.

## Как открыть локально

Откройте `index.html` в браузере.

## Как опубликовать на GitHub Pages

1. Загрузите содержимое этой папки в репозиторий.
2. В настройках репозитория откройте `Settings -> Pages`.
3. Выберите публикацию из ветки с файлом `index.html`.
4. После публикации GitHub Pages откроет прототип с главной страницы.
