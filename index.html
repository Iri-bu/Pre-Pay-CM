Предоплаты из OTA ?

i
Настройка определяет, создавать ли в WebPMS фактический платеж, когда канал присылает предоплату. Общее правило действует для объекта, а для отдельных OTA-каналов можно задать исключение.
Предоплата передается как фактический платеж
Используется как значение по умолчанию; для каждого канала правило можно изменить отдельно.

Исключение по каналу имеет приоритет над общим правилом объекта.
